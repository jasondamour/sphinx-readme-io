Getting Started
===============

This guide will help you get started with the project quickly.

Installation
------------

Install the package using pip:

.. code-block:: bash

   pip install my-project

Or with uv:

.. code-block:: bash

   uv add my-project

Quick Start
-----------

Here's a simple example to get you going:

.. code-block:: python

   from my_project import MyClass

   obj = MyClass()
   result = obj.do_something()
   print(result)

Next Steps
